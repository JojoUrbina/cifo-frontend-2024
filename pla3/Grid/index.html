La diferencia entre Flexbox y Grid

La principal diferencia entre Flexbox y Grid es como gestionan la distribución de los elementos dentro del diseño. Flexbox lo distribuye de manera unidimencional y Grid es un sistema bidimensional que permite la distribución en dos direcciones al mismo tiempo, en relacion a quien aparecio primero fue flexbox.

En cuanto al uso, Flexbox es excelente para diseños sencillos y unidimencionales, como barras de navegación horizontales o verticales o listas y Grid es más adecuado para diseños más diversos, como galerias, periodicos, revistas, con varias columnas y filas con difentes distribuciones.

Las funciones minmax() y repeat() se usan para establecer reglas flexibles y repetitivas, en caso de minmax() se podria usar para establecer el minimo y el maximo del tamaño de cada columna en grid y repeat() para establecer el tamaño y el numero de columnas en la que se dividira, tambien se puede usar para los degradados y otros valores que sean repetitivos.

La propiedad grid-template-columns se utiliza para establecer la distribucion y la cantidad de columnas que tendra el diseño, en caso de colocar 5 valores, se establecerian 5 columnas a las cuales se les puede aplicar diferentes tipos de medida, una muy frecuentemente usada es 1fr, que representa "una fraccion", tambien se pueden colocar las fracciones que sean necesarias.
