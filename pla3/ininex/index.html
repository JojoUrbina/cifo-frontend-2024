Diferencias entre los estilos CSS Inline, Internal, External.

Inline

Este es un parrafo aplicado con estilo CSS en linea.

Su estilo se limita a la etiqueta donde se aplica sin afectar los estilos de los otros elementos, se podra tomar como algo negativo o positivo dependiendo del problema a resolver.

Internal

Este es un parrafo con estilo aplicado con CSS Interno.

Se recomienda cuando se quiere cargar solo un archivo a la web y que los estilos esten aplicados solo a este documento.

Externo

Este es un parrafo con estilo aplicado con CSS Externo.

Es mas recomendable el CSS externo cuando se utiliza mas de una pagina web, facilita la organizacion y distribuye los estilos en todas las paginas necesarias aparte de poder reciclar de esta forma los estilos.
