Explicacion breve si la estructura de una pagina web debe ser de dos dimensiones

Una estructura con diferentes secciones es una buena idea para una pagina web dependiendo del objetivo de la misma, una landing page por ejemplo tiene una estructura similar, con menu superior, contenido principal, pero se evita de preferencia los menus laterales para maximizar la atencion del producto.

Si la estructura a la que se refiere es del tipo tabla, no es una buena idea en la acualidad definirla con esas estiquetas ya que existen diferentes formas de organizar contenido en una web como grid y flexbox.

Es entendible que en sus inicios era usado pero con tecnologias modernas no se deben limitar los estilos, hay casos en los que si pero con los avances y el desarrollo web no se aprovecharia al maximo sus herramientas.
