Enlaces

Funcionamiento del atributo "target"

El atributo target especifica dónde abrir el enlace cuando se hace click, entre sus valores tenemos:

  • _self: Abre el enlace en la misma ventana.
  • _blank: Abre el enlace en una nueva ventana.
  • _parent: Abre el enlace en el marco padre
  • _top: Abre el enlace en la ventana de nivel superior saliendo de todos los marcos.

Ejemplo:

Enlace a cifovirtual.cat (con target _blank)

Diferencia entre rutas absolutas y rutas relativas

  • Ruta absoluta: Especifica la ubicación completa del recurso desde la raíz del sistema de archivos o el dominio web.
  • Ruta relativa: Especifica la ubicación del recurso en relación con la ubicación del archivo que contiene el enlace.

Ejemplos:

Enlace relativo (ruta relativa)

Funcionamiento de la opción para enlazar directamente con una dirección de correo

El enlace con una dirección de correo electrónico utiliza el valor mailto: seguido de la dirección de correo electrónico.
